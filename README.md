个人使用(openwrt main源码)

构建r2s-ext4极简固件

仅包含passwall2(xray,sing-box),ttyd.

其他包:python3,bash,vim-full

默认旁路网关模式:192.168.27.5(主路由:192.168.27.1)

配置lan接口eth0,dnsmasq过滤广告,添加温控脚本
